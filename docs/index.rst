소개
----

암호화폐 트레이딩 시스템은 다음과 같은 주요 기능을 제공합니다:

* 실시간 시장 데이터 수집 및 분석
* 다양한 기술적 지표 기반의 매매 신호 생성
* 리스크 관리 및 포지션 사이징
* 백테스팅 및 성과 분석
* 텔레그램을 통한 실시간 알림
* 웹 기반 대시보드

시스템 아키텍처
~~~~~~~~~~~~~

.. image:: _static/architecture.png
   :alt: 시스템 아키텍처
   :align: center

주요 컴포넌트:

1. **데이터 수집기**
   - 실시간 시장 데이터 수집
   - 기술적 지표 계산
   - 데이터 저장 및 관리

2. **전략 엔진**
   - 매매 신호 생성
   - 포지션 관리
   - 리스크 관리

3. **실행 엔진**
   - 주문 실행
   - 포지션 추적
   - 성과 분석

4. **알림 시스템**
   - 텔레그램 봇 연동
   - 실시간 알림
   - 성과 보고

5. **웹 대시보드**
   - 실시간 모니터링
   - 성과 시각화
   - 설정 관리

시스템 요구사항
~~~~~~~~~~~~~

* Python 3.8 이상
* PostgreSQL 12 이상
* Redis 6 이상
* 텔레그램 봇 토큰

성능 지표
~~~~~~~~

* 실시간 데이터 처리: < 100ms
* 주문 실행: < 500ms
* 백테스팅 속도: 100x 실시간
* 메모리 사용량: < 1GB
* CPU 사용률: < 50%
